# Go Global Invitation

Prototype for THG × OnPoint event invitation.

## Features

- Interactive passport-style invitation card
- 3D hover effects
- Modal popup with event details
- Responsive design

## Project Structure

```
go-global-invitation/
├── index.html      # Main HTML file
├── css/
│   └── style.css   # Styling
├── js/
│   └── script.js   # Interactions
└── README.md       # Documentation
```

## Getting Started

Open `index.html` in your browser to view the invitation.
